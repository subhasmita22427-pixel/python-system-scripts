# Search and Sort Utility

## Overview
This project is a command-line utility that performs various operations on a list of numbers.
It is designed to strengthen logical thinking and problem-solving using basic Python.

## Features
- Accepts only numeric input from the user
- Searches for a number in the list
- Sorts numbers in ascending and descending order
- Displays even and odd numbers separately
- Finds multiples of a user-specified number
- Handles invalid input gracefully

## Why This Project
Searching, sorting, and filtering data are common operations in real-world applications.
This project focuses on understanding the logic behind these operations instead of using built-in shortcuts.
